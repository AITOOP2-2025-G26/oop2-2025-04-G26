## 音声文字起こしプログラム

1. リポジトリの目的
- このプログラムでは、10秒間音声を録音し，録音された内容を文字起こしして，テキストファイルに保存することを目的とする。
2. 実行するのに必要なモジュール
- requirments.txt

3. 実行の流れ
- 実行をすると、音声の録音が始まる。その録音された内容を文字で起こして、テキストで出力する。
